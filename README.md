# 卫星仿真 (Satellite Sim)

晨昏轨道卫星扫描动画：地固系地球（不自转）、默认中国视角、Wayfinder 风格时间控制、30 天条带褪色。

## 快速开始

```bash
cp .env.example .env   # 填入 VITE_CESIUM_ION_TOKEN
npm install
npm run dev
```

## 时间控制（底部）

| 按钮 | 作用 |
|------|------|
| **LIVE** | 回到真实 UTC，实时播放（1×） |
| **▶ / ⏸** | 播放 / 暂停 |
| **1×** | 倍速 1（3600× 模拟，约 1 小时/秒） |
| **2×** | 倍速 2（86400× 模拟，约 1 天/秒，用于看褪色） |
| **时间轴** | 拖动查看任意时刻 |

## 核心设计

- **轨道**：500 km 太阳同步圆轨道，地固系固定轨道环，周期 ~94 分钟
- **地球**：纹理不自转；默认显示中国；可手动旋转地球（相机）
- **条带**：20 km 宽，白色 → 灰色 → 消失（30 天）
- **时间轴**：只控制模拟时间流速，不改变轨道参数

## 修改参数

编辑 `src/config/satellite.js`
